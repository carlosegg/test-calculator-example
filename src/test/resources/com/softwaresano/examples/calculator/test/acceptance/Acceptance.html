Calculadora

Cuando el usuario introduce dos números no negativos, se calcula la suma de ambos números. Ejemplo: Si el usuario introduce 2 y 2 el resultado debe ser 2 + 2 = 4

Suma

Cuando el usuario introduce como primer sumando un 2 y como segundo sumando un 2. El resultado debe ser: 2 + 2 = 4

Cuando el usuario introduce como primer sumando un número negativo -1 y como segundo sumando un 2 ERROR: Los dos sumandos han de ser positivos

Cuando el usuario introduce como primer sumando un número negativo -1 y como segundo sumando otro numero negativo -2 ERROR: Los dos sumandos han de ser positivos

Cuando el usuario introduce como primer sumando 1 y como segundo sumando un número negativo -2 ERROR: Los dos sumandos han de ser positivos
